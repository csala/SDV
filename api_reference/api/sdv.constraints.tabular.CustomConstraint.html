sdv.constraints.tabular.CustomConstraint¶

class sdv.constraints.tabular.CustomConstraint(transform=None, reverse_transform=None, is_valid=None)[source]¶

Custom Constraint Class.

This class simply takes the transform, reverse_transform and is_valid methods as optional arguments, so users can pass custom functions for each one of them.

Parameters

  • transform (callable) – Function to replace the transform method.

  • reverse_transform (callable) – Function to replace the reverse_transform method.

  • is_valid (callable) – Function to replace the is_valid method.
