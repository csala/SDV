sdv.sampler.Sampler.sample_all¶

Sampler.sample_all(num_rows=None, reset_primary_keys=False)[source]¶

Sample the entire dataset.

sample_all returns a dictionary with all the tables of the dataset sampled. The amount of rows sampled will depend from table to table, and is only guaranteed to match num_rows on tables without parents.

This is because the children tables are created modelling the relation that they have with their parent tables, so its behavior may change from one table to another.

Parameters

  • num_rows (int) – Number of rows to be sampled on the first parent tables. If None, sample the same number of rows as in the original tables.

  • reset_primary_keys (bool) – Whether or not reset the primary key generators.

Returns

A dictionary containing as keys the names of the tables and as values the sampled datatables as pandas.DataFrame.

Return type

dict
